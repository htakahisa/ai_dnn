# 分割版 run_game

同じフォルダへ以下の6ファイルを置き、`run_game.py`を実行してください。

- `run_game.py`: 起動、初期化、ラウンド生成
- `game_core.py`: 定数、データ読み込み、Character
- `combo_awakening.py`: コンボ、覚醒、告知
- `abilities_los.py`: スモーク、射線、フラッシュ、リコン
- `battle_logic.py`: 移動、射撃、スパイク、勝敗
- `rendering_ui.py`: Tkinter描画、クリック操作

既存の `controllers.py`、`learning_defender.py`、`map_data.py`、`roster_select.py`、
`character_stats.py`、`player_combos.py`、`awakening_events.py` は同じフォルダに必要です。
